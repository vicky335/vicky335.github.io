Markdown入门

Jun 18, 2017 #markdown

Markdown 是一种“轻量级标记式语言”，创始人为约翰·格鲁伯（John Gruber）。这个语言的目的是希望大家使用“易于阅读、易于撰写的纯文字格式，并选择性的转换成有效的 XHTML (或是HTML)”。

介绍

Markdown 是一种“轻量级标记式语言”，它的语法简单，不像一般的的文字处理软件Word或Pages有大量的排版、字体设置。它使我们只专心于文字和内容，用“标记式语法” 来代替常见的排版格式。从内容到格式，甚至插图，都是键盘就可以通通搞定了。

目前支持Markdown语法的编辑器和网站都好多。例如：MacDown、typora、Mou、GitHub、简书等等。

Markdown能导出多种格式，可以导出HTML格式的文件用来网站发布，也可以导出PDF格式。

Markdown语法参考：

  • 创始人约翰·格鲁伯（John Gruber）的Markdown语法
  • Markdown中文版语法

特点

  1. 纯文本，兼容性极强，可以用所有文本编辑器打开
  2. Markdown专注于文字和内容，而不是排版，能让我们安心写作
  3. Markdown的语法简单，有极好的可读性，常用的标记符号不超过十个
  4. 格式转换方便，除了本身的.md文件外，还能轻松转换为HTML、PDF等格式
  5. 如果你是爱折腾的人，Markdown还能支持流程图、复制的表格、公式等等

语法规则

代码区域

对于程序员，文章中有代码是肯定的。我们需要代码框把代码引起来，这个时候，我们可以用3个`把代码引起来就可以了，

var a =3;
int b = 33;

    写法：
    

标题

可以在标题内容前输入特定数量的#号来实现对应级别的HTML样式标题。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

一级和二级标题还可以写成：

一级标题
===================

二级标题
--------------------

Markdown会转成：

<h1>一级标题</h1>
<h2>二级标题</h2>
<h6>六级标题</h6>

列表

我们知道列表有 有序列表 和 无序列表 两种。在Markdown下，只需要在文字前加上 - 或 * 即变为无序列表，而有序列表则是直接在文字前面加1. 2. 3.。

* 列表1
* 列表2
* 列表3

- 列表1
- 列表2
- 列表3

1. 列表1
2. 列表2
3. 列表3

Markdown会转成：

<ol>
    <li>列表1</li>
    <li>列表2</li>
    <li>列表3</li>
</ol>

<ul>
    <li>列表1</li>
    <li>列表2</li>
    <li>列表3</li>
</ul>

注意：符号与文字之间要加上一个字符的空格

引用

引用只需要在内容段落开头前加上右尖括弧(>)就可以了。可以是开头加或每行加，效果都是一样。

> 这一整段的内容都会作为一个HTML的引用元素。
引用元素是会自动优化排版的（reflowable，可回流）。

或

> 这一整段的内容都会作为一个HTML的引用元素。
> 引用元素是会自动优化排版的（reflowable，可回流）。

Markdown会转成：

<blockquote><p>这一整段的内容都会作为一个HTML的引用元素。引用元素是会自动优化排版的（reflowable，可回流）。</p></blockquote>

如果是引用中的引用，可以同两个(>)开头，如此类推，即可实现多级引用。

> 这一整段的内容都会作为一个HTML的引用元素。
>> 引用元素是会自动优化排版的（reflowable，可回流）。

换行

单一段落

<p>

多行

<br />

如果你真的想在Markdown中插入换行标签
，你可以在行尾输入两个或以上的空格，然后Enter。 这样插入换行十分麻烦，但是“每个换行都转换为
”在 Markdown中并不合适，所以只在你确定你需要时手动添加。

连结

  • 文字连结。只需要使用[连结文字](连结地址)，如
[Markdown](http://zh.wikipedia.com/wiki/Markdown "Markdown")

显示效果：Markdown

  • 索引连结。连结地址可以放在段落后面的注脚，前面放上连结参照标签区分。如，先在内容行内插入
[连结文字][连结引用标签]

然后在段落后面（或文件结尾）放上以下内容，就可以生成连结：

[连结引用标签]：连结地址 "连结标题"
  • 自动连结。Markdown还支持以比较简短的自动连结形式来处理网址和电子邮件，只要使用尖括号<>包起来，就会自动把它转成连结。如
<http://www.example.com>
<address@example.com>

显示效果：http://www.example.com address@example.com

此时，Markdown会转为：

<a href="http://example.com/">http://example.com/</a>

图片

插入图片，只需要使用![替代文字(可省略)](图片连结地址 "title文字")即可。不过目前Markdown还不能指定图片的宽高，如有需要，可以使用普通的<img>标签。

强调

  • 粗体

    使用2个**或用2个__，把要加粗的内容包住，如：

    **粗体**
    
__粗体__

  • 斜体

    使用2个**或用2个__，把需要显示斜体的内容包住，如：

    *斜体*
    
_粗体_

  • 粗斜体

    使用3个*** 或用2个___，把要加粗斜体的内容包住，如：

    ***斜体***
    
___粗体___

  • 转义

    Markdown可以利用反斜杠来插入一些在语法中有特殊意义的符号，例如想显示*，就在星号前加上反斜杠\*。 Markdown支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

        \\    反斜线
    \`    反引号
    \*    星号
    \_    下划线
    \{\}  大括号
    \[\]  中括号
    \(\)  小括弧
    \#    井号
    \+    加号
    \-    减号
    \.    英文句号
    \!    感叹号
